# 明日方舟人事部档案

本项目是一个用于自动化获取、存储、管理和可视化明日方舟（Arknights）游戏玩家寻访（抽卡）历史记录的软件系统。

目前实现了核心的数据获取功能，提供了完整的自动化流程和模块化架构。

## 功能

- **自动获取**: 通过模拟登录，自动从官方服务器获取寻访记录。
- **本地存储**: 将寻访记录以 JSON 格式安全地存储在本地。
- **增量更新**: 能够智能地合并新旧数据，保留完整的寻访历史。
- **安全设计**: 用户凭证经过加密处理，保障账户安全。
- **模块化架构**: 采用专家系统模式，各功能模块独立且易于维护。

## 项目架构

```
Arknights-HR-Archives/
├── update_gacha_data.py      # 主程序入口
├── requirements.txt          # 项目依赖
├── config/
│   └── system.json          # 系统配置文件
├── solvers/                 # 核心功能模块
│   ├── authenticator.py     # 身份认证模块
│   ├── credential_manager.py # 凭证管理模块
│   ├── gacha_data_fetcher.py # 数据获取模块
│   └── gacha_data_storer.py  # 数据存储模块
└── users/                   # 用户数据目录
    └── [用户名]/
        └── accounts/
            └── [游戏UID]/
                ├── config.json    # 账户配置
                ├── data.json      # 寻访记录数据
                └── metadata.json  # 元数据信息
```

## 如何运行

### 1. 安装依赖

首先，请确保您已经安装了 Python 3.x 环境。然后，在项目根目录下打开终端，运行以下命令安装所需的依赖库：

```bash
pip install -r requirements.txt
```

### 2. 配置您的账号信息

程序需要使用您的明日方舟账号信息来进行真实的认证和数据获取。

1.  创建用户目录结构（以用户名 `test` 为例）：
    ```bash
    mkdir -p users/test/accounts
    ```

2.  创建账户配置文件 `users/test/accounts/test_account/config.json`，并填入您的真实信息：

    ```json
    {
      "username": "您的手机号",
      "password": "您的登录密码"
    }
    ```

    **请注意**：您的密码在首次运行时会被脚本自动加密，并覆盖原有的明文密码。后续运行将直接使用加密后的凭证。

### 3. 运行主程序

完成以上配置后，在项目根目录下运行主程序：

```bash
python update_gacha_data.py users/test/accounts/test_account/config.json test
```

**参数说明**：
- 第一个参数：账户配置文件的路径
- 第二个参数：系统用户ID，用于创建目录结构

程序将自动执行以下操作：

- **凭证管理**: 加载并加密您的凭证。
- **身份认证**: 使用您的凭证进行链式令牌交换，获取访问权限。
- **数据获取**: 获取您的寻访记录。
- **数据存储**: 将获取到的记录增量保存到本地。

### 4. 查看结果

运行成功后，您的寻访数据将被保存在以下目录中：

`./users/test/accounts/<您的游戏UID>/`

该目录下会生成以下文件：

- `config.json`: 存储加密后的账户凭证。
- `data.json`: 存储格式化后的寻访记录。
- `metadata.json`: 存储更新时间、记录总数等元数据。

您现在可以自由地使用这些数据进行分析或二次开发。

## 核心模块说明

### Authenticator（身份认证专家）
负责处理完整的身份验证流程，包括：
- 初始令牌获取
- 应用令牌交换
- 角色绑定和登录
- 用户目录创建

### GachaDataFetcher（数据获取专家）
负责从官方服务器获取寻访记录，支持：
- 分页获取大量数据
- 多种寻访类型
- 数据格式化

### GachaDataStorer（数据存储专家）
负责本地数据管理，提供：
- 增量数据合并
- JSON格式存储
- 元数据管理

### CredentialManager（凭证管理专家）
负责账户凭证的安全管理，包括：
- 凭证加载和加密
- 令牌持久化存储
- 安全的凭证更新

## 技术特性

- **安全性**: 使用cryptography库进行凭证加密
- **可靠性**: 完整的错误处理和日志记录
- **可扩展性**: 模块化设计，易于添加新功能
- **自动化**: 一次配置，自动完成全部流程

## 注意事项

1. **账户安全**: 您的账号信息仅存储在本地，经过加密处理。
2. **网络要求**: 需要能够访问明日方舟官方API服务器。
3. **使用频率**: 请合理使用，避免过于频繁的请求。
4. **数据备份**: 建议定期备份重要的寻访记录数据。
