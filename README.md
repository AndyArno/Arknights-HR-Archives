# 明日方舟寻访记录档案系统

本项目是一个用于自动化获取、存储、管理和可视化明日方舟（Arknights）游戏玩家寻访（抽卡）历史记录的软件系统。

## 功能

- **自动获取**: 通过模拟登录，自动从官方服务器获取寻访记录。
- **本地存储**: 将寻访记录以 JSON 格式安全地存储在本地。
- **增量更新**: 能够智能地合并新旧数据，保留完整的寻访历史。
- **安全设计**: 用户凭证经过加密处理，保障账户安全。

## 如何运行测试脚本

本项目提供了一个端到端的测试脚本，用于验证核心的数据获取和存储功能。请按照以下步骤操作：

### 1. 安装依赖

首先，请确保您已经安装了 Python 3.x 环境。然后，在项目根目录下打开终端，运行以下命令安装所需的依赖库：

```bash
pip install -r requirements.txt
```

### 2. 配置您的账号信息

测试脚本需要使用您的明日方舟账号信息来进行真实的认证和数据获取。

1.  找到测试用户配置文件：`./users/test/accounts/test_account/config.json`
2.  打开该文件，将其中的占位符替换为您的真实信息：

    ```json
    {
      "username": "您的手机号",
      "password": "您的登录密码"
    }
    ```

    **请注意**：您的密码在首次运行时会被脚本自动加密，并覆盖原有的明文密码。后续运行将直接使用加密后的凭证。

### 3. 运行测试脚本

完成以上配置后，在项目根目录下运行测试脚本：

```bash
python test/test_stage_one.py
```

脚本将自动执行以下操作：

- **凭证管理测试**: 加载并加密您的凭证。
- **身份认证测试**: 使用您的凭证进行链式令牌交换，获取访问权限。
- **数据获取测试**: 获取您的寻访记录。
- **数据存储测试**: 将获取到的记录增量保存到本地。

### 4. 查看结果

测试成功后，您的寻访数据将被保存在以下目录中：

`./users/test/accounts/<您的游戏UID>/`

该目录下会生成两个文件：

- `data.json`: 存储格式化后的寻访记录。
- `metadata.json`: 存储更新时间、记录总数等元数据。

您现在可以自由地使用这些数据进行分析或二次开发。
